# v1.1.0

**Release Date:** 2026-02-02

## New Features

### Location Support
- Add location to memories when capturing
- Auto-detect current location
- Manual location selection via search or map
- View and edit location in memory details
- Tap location to open in Maps app

### CloudKit Sync
- Memories now sync to iCloud automatically
- Data persists even after app deletion
- Sync across multiple devices with same Apple ID

### Data Storage Warning
- Added warning in Settings about data storage
- Users are informed about iCloud sync for data backup

## Improvements

### Notification Messages
- "Document this moment" → "Keep this moment"
- "Time to save a memory" → "Time to make a memory"

### Share
- Simplified sharing to export image only (no text metadata)

### Input Validation
- Note input now enforces 280 character limit

## Technical

- Added `LocationService` for location handling
- Added `LocationPickerView` for location selection UI
- Extended `Memory` model with `latitude`, `longitude`, `locationName` fields
- Added default values to all model properties for CloudKit compatibility
- Enabled CloudKit database sync in SwiftData container
- Added PR template

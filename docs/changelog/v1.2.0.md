# v1.2.0

**Release Date:** 2026-02-03

## New Features

### Korean Language Support
- Full Korean localization
- In-app language selection (Settings > Language)
- Dropdown picker for easy language switching

### Data Export
- Export all memories as JSON file
- Includes image data, notes, location, and timestamps

## Improvements

### Notifications
- Minimum 30-minute gap between notifications
- Prevents notification fatigue from too frequent prompts

### Notification Tap Behavior
- Camera now opens immediately when tapping notification
- Works even when app was completely terminated (cold launch)

## Technical

- Added `Localizable.xcstrings` with Korean translations
- Added `shouldOpenCamera` flag in `AppDelegate` for cold launch handling
- Added `exportData()` function in `SettingsView`
- Updated `NotificationService` with 30-minute minimum gap logic
